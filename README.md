# Exercício - Idade

Este repositório contém um exercício simples de programação para manipular e exibir dados em Kotlin

##  Objetivo

Praticar lógica de programação e manipulação de dados através de um programa que trabalha com idade.

##  Tecnologias utilizadas

* Kotlin
* Android Studio IDE
* Conceitos básicos de lógica de programação

## Como executar

1. Clone este repositório:

```
git clone https://github.com/Luana598/Exercicio_idade.git
```

2. Acesse a pasta do projeto:

```
cd Exercicio_idade
```

3. Execute o arquivo principal conforme a linguagem utilizada.
